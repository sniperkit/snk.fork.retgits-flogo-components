# Jira-Query Ticket
This activity provides your flogo application to query in JIRA domain.

## Installation

```bash
flogo install github.com/sniperkit/snk.fork.retgits-flogo-components/activity/jira/queryticket
flogo ensure
```

Link for flogo web:
